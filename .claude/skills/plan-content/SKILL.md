## 1단계: 타겟 페르소나 설정

기획 전 아래 항목을 명확히 정의하세요.

| 항목 | 내용 |
|------|------|
| 연령대 | 예: 30대 초반 엄마 |
| 자녀 나이 | 예: 7세 남아 |
| 핵심 고민 | 예: 아이가 학교에서 집중을 못함, 산만하다는 피드백 받음 |
| 이미 시도한 것 | 예: 오메가3 먹여봤지만 효과 없음 |
| 정보 수집 채널 | 예: 맘카페, 인스타그램 육아 계정 |
| 발작버튼 키워드 | 예: 산만함, 집중력, ADHD, 불안 |

**발작버튼 키워드**란 타겟 페르소나가 봤을 때 즉각 반응하는 감정적 트리거 단어입니다. 소비자 인사이트 파일에서 추출하거나, 제품 카테고리에 맞게 선정하세요.

## 2단계: 재정의 (Reframing) 전략 수립

광고의 핵심은 소비자의 문제를 **우리 제품만 풀 수 있는 방식으로 재정의**하는 것입니다.

### 재정의 3가지 기법

**① 문제 재정의**
- 소비자가 인식하는 문제(표면 문제)를 더 근본적인 원인으로 전환
- 예: "아이가 산만하다" → "스트레스 → 코르티솔 과다 → 세로토닌·도파민 불균형 → 집중력·감정 조절 어려움"
- 근본 원인을 해결하는 제품임을 강조

**② 경쟁 비교 재정의**
- 기존에 시도한 해결책의 한계를 지적하고 우리 제품의 차별점 제시
- 예: "일반 오메가3만으로는 부족한 이유 — 뇌 영양소 밸런스가 핵심"

**③ 성분/기능 재정의**
- 성분명을 소비자 언어로 번역하여 가치를 재정의
- 예: "테아닌 = 아이 뇌가 스트레스를 스스로 조절하는 힘"

각 컨셉에 어떤 재정의 기법을 쓸지 `reframe_type` 필드에 명시하세요.

## 3단계: 핵심 키워드 선정

전체 캠페인에서 사용할 **핵심 키워드 5~10개**를 선정합니다.
- 키워드는 콘텐츠 간 중복 사용 가능 (일관성 유지)
- 발작버튼 키워드 + 성분 키워드 + 혜택 키워드 조합

예시:
```
발작버튼: 집중력, 산만함, 예민함, 불안
성분: 테아닌, 가바, 오메가3, 유산균
혜택: 뇌 영양, 감정 안정, 공부 집중, 수면
```

## 4단계: 컨셉 설계 원칙

### 컨셉 앵글 (각 컨셉마다 하나 선택)

15개 컨셉은 아래 앵글을 고루 분산하여 서로 겹치지 않게 설계:

| 앵글 | 설명 | 적합한 타겟 심리 |
|------|------|----------------|
| **페인포인트** | 부모의 핵심 고민을 직접 건드림 | 문제를 인식하고 있으나 해결책 못 찾음 |
| **혜택형** | 제품 사용 후 기대되는 구체적 변화 강조 | 효과를 먼저 보고 싶어하는 타겟 |
| **성분강조** | 핵심 성분의 과학적 기능성 부각 | 정보 탐색형, 성분 따지는 타겟 |
| **라이프스타일** | 건강하고 행복한 아이의 일상 묘사 | 이상적인 육아 이미지에 공감하는 타겟 |
| **신뢰/안심** | 성분 안전성, 인증, 전문가 추천 | 안전성 걱정이 구매 장벽인 타겟 |
| **편의성** | 간편한 섭취, 아이가 스스로 먹음 | 바쁜 부모, 편의 우선 타겟 |
| **감성소구** | 부모의 사랑과 아이의 행복 연결 | 감성적 연결을 통한 브랜드 호감 |
| **비교형** | 기존 제품/방법 대비 차별점 명확화 | 여러 제품 써봤지만 효과 없었던 타겟 |
| **역발상형** | 상식을 뒤집는 의외의 접근 | 광고 피로도 높은 타겟, 주목도 우선 |
| **공포/손실형** | 지금 안 하면 생기는 문제 강조 (FOMO) | 불안 심리가 높은 타겟 |
| **사회증거형** | 다른 엄마들의 사용 후기, 공통 경험 | 주변 반응에 민감한 타겟 |
| **권위형** | 전문가·연구·기관 인용으로 신뢰 구축 | 근거 기반 의사결정 타겟 |
| **정보제공형** | 유용한 정보를 제공하며 자연스럽게 제품 연결 | 정보 콘텐츠로 진입하는 타겟 |
| **할인정보형** | 현재의 할인 혜택을 강조 | 구매를 적극 고민 중인 타겟 |

### 콘텐츠 뼈대: HEAD-BODY-LEGS 구조

모든 영상 광고는 이 3파트로 구성:

```
HEAD (1~3초): 시청 이유 제공 — 스크롤을 멈추게 하는 훅
  → 발작버튼 키워드 또는 강렬한 장면으로 즉각 주목
  → "이게 왜 나 얘기지?" 반응 유도

BODY (5~20초): 구매 이유 제공 — 설득의 본론
  → 문제 재정의 → 성분/근거 → 변화/결과
  → 감정 곡선: 공감 → 희망 → 확신

LEGS (1~3초): 행동 유도 — CTA
  → 명확한 다음 행동 (지금 확인, 무료 샘플, 링크 클릭)
  → FOMO 또는 혜택 언급
```

### 콘텐츠 아웃라인 유형

각 영상 컨셉에 아래 유형처럼 아웃라인을 정해주세요. 나열되지 않은 유형도 가능하지만, **훅 + 제품제시 + 사용후반응**은 반드시 포함:

| 유형 | 구성 순서 | 특징 |
|------|----------|------|
| **A형** | 훅→문제→재정의→제품→성분→결과→CTA | 정보형, 신뢰 구축 |
| **B형** | 훅→공감→문제→제품→후기→CTA | 공감 우선, 소셜 프루프 |
| **C형** | 훅→B&A→이유→제품→CTA | 변화 먼저 보여주기 |
| **D형** | 훅→숫자/통계→문제→제품→근거→CTA | 데이터 기반 설득 |
| **E형** | 훅→역발상→재정의→제품→CTA | 충격+반전, 주목도 극대화 |
| **F형** | 훅→스토리→감정고조→제품→CTA | 내러티브형, 감성 소구 |
| **G형** | 훅→비교→차별점→제품→CTA | 경쟁 비교형 |
| **H형** | 훅→정보→팁→제품→CTA | 정보제공형, 네이티브 광고 스타일 |

## 5단계: 프로덕션 스펙 설계

### 등장 형식 (appearance_type)
- `사람등장형`: 실제 부모/아이 등장, 공감과 사실감 강조
- `제품중심형`: 제품만 등장, 성분/기능 강조에 적합
- `만화/일러스트형`: 캐릭터 애니메이션, 아이 타겟 또는 가벼운 톤
- `모션그래픽형`: 텍스트+그래픽, 정보 전달에 특화
- `혼합형`: 실사+그래픽 합성

### 나레이션 (narration)
- `없음`: 자막만, BGM만
- `여성VO_따뜻한`: 공감적·감성적 내용에 적합
- `여성VO_전문적`: 성분·근거 설명에 적합
- `남성VO_신뢰감`: 권위형·전문가 추천에 적합
- `아이목소리`: 감성·공감 극대화
- `텍스트자막전용`: 무음 환경 최적화

### 영상 길이 (duration_sec)
- `6초`: 제품 정보 요약 전달 또는 할인 정보 전달
- `15초`: 인스타그램 스토리/릴스 최적화, 임팩트 집중
- `30초`: 표준 피드 광고, 설득 구조 완성
- `60초`: 정보량 많은 컨셉, 스토리텔링형

## 6단계: 시청 이유 & 구매 이유 체크

각 컨셉 설계 시 아래 항목을 최소 2개씩 충족하는지 확인:

**시청 이유 (훅의 질)**
1. 뻔하지 않은 이야기 (의외성)
2. 실용적 이익 기대 (정보/팁)
3. 몰입형 포맷 (감정이입 가능한 상황)
4. 시각적 차별화 (눈에 띄는 비주얼)
5. 3초 이내의 짧은 메시지

**구매 이유 (설득의 질)**
1. 신뢰/믿음 (성분 근거, 인증, 후기)
2. 독보적 차별성 (이것만의 특징)
3. 개선 확신 (변화가 그려짐)
4. 손실 회피/FOMO (지금 안 사면 손해)
