## 0. 레퍼런스 학습 인사이트 우선 적용 [필수]

카피/스크립트 작성 전 반드시 `brand/reference_learnings.md`를 읽고 아래 원칙을 따릅니다.

### 학습 적용 기준 (업데이트됨)

**말투·문장구조·설득흐름·어미·훅 방식**
→ 자사·타사 구분 없이 37개 전체 영상 기반 적용
→ `brand/reference_learnings.md` Section A 참고

**키워드·실제 워딩·설득논리·문제제기·소구점·감정**
→ 제품별로 검증된 표현을 그대로 사용 가능
→ `brand/reference_learnings.md` Section B 해당 제품 참고

| 제작할 제품 | 참고할 섹션 |
|---|---|
| 뉴턴젤리 | Section B-1 |
| 유산균 | Section B-2 |
| 오메가3 | Section B-3 |

**콘텐츠 유형별 구조**
→ 의뢰받은 유형에 맞는 세부 패턴 적용
→ `brand/reference_learnings.md` Section C 해당 유형 참고

### 카피 작성 시 체크리스트

- [ ] `brand/reference_learnings.md` 읽었는가?
- [ ] 해당 제품(B섹션)의 검증된 키워드·표현을 후킹 또는 본문에 포함했는가?
- [ ] 콘텐츠 유형(C섹션)에 맞는 구조와 어미를 적용했는가?
- [ ] 이전 캠페인과 동일한 구조·표현이 반복되지 않는가?
- [ ] 흘러가는 자막이 호흡 단위로 나뉘어 있는가? (레퍼런스 문체 기준)

## 1. 후킹 문구 (Hook Copy)

광고 첫 1~3초, 스크롤을 멈추게 하는 문장. "이게 나 얘기네" 반응을 유도하는 것이 목표.

### 후킹 문구 작성 원칙
- **최대 15자 이내** (모바일 피드 최적화)
- 발작버튼 키워드 또는 타겟 상황 직접 언급
- 구체적인 숫자·수치 포함 권장
- 뻔하지 않은 의외성, 또는 강한 공감 상황으로 시작
- 질문형 또는 부정 카피로 호기심 유발 가능

### 후킹 문구 유형

| 유형 | 예시 | 사용 상황 |
|------|------|----------|
| **타겟 피셔형** | "7살 아이 집중력 걱정하는 엄마" | 특정 페르소나를 콕 집어 부름 |
| **숫자형** | "아이 집중력, 3주 만에 달라졌어요" | 구체적 변화·기간으로 신뢰 |
| **부정 카피형** | "오메가3 혼자 먹이면 안 되는 이유" | 반전·금지 메시지로 클릭 유도 |
| **질문형** | "우리 아이 왜 이렇게 산만하지?" | 부모의 내면 독백 그대로 반영 |
| **B&A형** | "매일 싸우던 숙제, 이제 스스로 해요" | 변화 전후 대비로 희망 제시 |
| **공포/손실형** | "초등 첫 학기, 집중력 골든타임 놓치면" | FOMO 활용, 긴박감 조성 |
| **비밀공개형** | "소아과 의사가 추천하는 뇌 영양소" | 권위·전문 정보로 클릭 유도 |
| **사회증거형** | "맘카페 3만 명이 찾는 그 영양제" | 대중 검증으로 안심 |
| **역발상형** | "열심히 공부시켜도 집중 못 하는 진짜 이유" | 상식 뒤집기로 주목도 극대화 |

## 2. 본문 카피 (Body Copy)

설득의 본론. 감정 곡선: **공감 → 희망 → 확신** 순서로 전개.

### 본문 카피 작성 원칙
- 핵심 성분 1~2가지 언급
- 재정의 로직 기반으로 문제를 우리 제품이 풀 수 있는 방식으로 전환
- 변화·결과가 구체적으로 그려지도록 작성

### 재정의(Reframing) 카피 3기법

**① 문제 재정의**
소비자가 인식하는 표면 문제를 근본 원인으로 전환
```
"아이가 산만한 게 아니라 → 스트레스 → 코르티솔 과다 → 뇌 영양 불균형"
→ 카피: "산만한 게 아니에요. 뇌에 필요한 게 부족한 거예요."
```

**② 경쟁 비교 재정의**
기존 해결책의 한계 지적 → 우리 제품 차별점 제시
```
"오메가3만으로는 부족해요. 뇌 밸런스엔 테아닌+가바도 필요합니다."
```

**③ 성분/기능 재정의**
성분명을 소비자 언어로 번역
```
테아닌 → "아이 뇌가 스트레스를 스스로 조절하는 힘"
가바 → "흥분한 신경을 진정시키는 자연 안정제"
```

## 3. CTA (행동 유도 문구)

- "지금 확인하기", "불만족시 100% 환불", "구매하러 가기"
- 2주만에 아이 바꾸기
- 긴박감 조성: "오늘만 할인", "한정 수량" (사실인 경우만)
- FOMO 활용: "이번 주만 첫 구매 50% 할인"

## 4. GFA 25가지 카피 기법

각 컨셉 앵글에 맞게 아래 기법 중 1~2개를 선택해 적용:

| # | 기법명 | 설명 | 예시 |
|---|--------|------|------|
| 1 | **B&A (Before & After)** | 변화 전후를 대비시켜 희망 제시 | "산만했던 아이 → 집중해서 숙제 끝냄" |
| 2 | **숫자 강조** | 구체적 수치로 신뢰·신속성 부여 | "3주 만에", "97% 만족", "성분 함량 2배" |
| 3 | **소셜 프루프** | 다른 부모들의 공통 경험·선택 강조 | "맘카페 1위", "10만 엄마의 선택" |
| 4 | **긴박감/희소성** | 지금 행동하지 않으면 손해라는 압박 | "오늘까지 50% 할인", "재입고 미정" |
| 5 | **부정 카피** | 금지·경고 메시지로 역설적 클릭 유도 | "이것만은 하지 마세요", "먹이면 안 되는 영양제" |
| 6 | **타겟 피셔** | 특정 페르소나를 직접 호명 | "7살 아이 둔 엄마라면", "초1 입학 앞둔 부모님께" |
| 7 | **할인/이벤트** | 가격 혜택을 전면에 배치 | "첫 구매 반값", "3개 사면 1개 무료" |
| 8 | **리뷰 강조** | 실사용자 후기 직접 인용 | '"4주 먹이고 선생님한테 칭찬 받았어요"' |
| 9 | **USP 강조** | 경쟁사 대비 독보적 차별점 | "국내 유일 테아닌+가바+오메가3 복합" |
| 10 | **비밀공개** | 알려지지 않은 정보 제공하는 척 | "소아과 의사가 말 안 해주는 뇌 영양 비밀" |
| 11 | **가치 비교** | 가격 대비 가치를 친숙한 단위로 환산 | "하루 커피 한 잔 값으로 아이 뇌 영양 챙기기" |
| 12 | **질문형** | 부모의 내면 고민을 질문으로 표현 | "우리 아이만 왜 이렇게 산만할까요?" |
| 13 | **친근한 문체** | 이웃 엄마가 말하듯 편안한 말투 | "저도 처음엔 반신반의했는데요..." |
| 14 | **점층적 긴박감** | 단계적으로 위기감을 높여 결국 CTA | "1학년 → 2학년 → 격차는 점점 벌어집니다" |
| 15 | **권위 인용** | 전문가·연구·기관 데이터 활용 | "서울대 연구팀 발표", "소아과 전문의 추천" |
| 16 | **공감 상황 묘사** | 타겟의 일상 장면을 구체적으로 묘사 | "숙제 10분에 한 번씩 딴짓하는 아이" |
| 17 | **시각적 후킹** | 카피 없이 장면으로 시선 고정 | 아이 집중 vs 산만 대비 장면 |
| 18 | **스토리텔링** | 상황 → 갈등 → 해결 서사 구조 | "늘 싸우다가 → 이걸 알게 되고 → 지금은" |
| 19 | **실패 경험 공유** | 비슷한 실패를 먼저 고백해 공감 획득 | "저도 오메가3만 먹였다가 실망했어요" |
| 20 | **성분 과학화** | 성분을 쉬운 과학 언어로 설명 | "테아닌 = 뇌 알파파 증가 → 차분한 집중" |
| 21 | **FOMO** | 안 하면 손해·뒤처진다는 심리 자극 | "초등 골든타임, 지금이 마지막 기회" |
| 22 | **역발상** | 상식을 뒤집는 의외의 주장 | "공부를 못 시켜서가 아니에요" |
| 23 | **편의성 강조** | 간편 섭취, 아이가 좋아하는 맛 부각 | "젤리처럼 씹어 먹는 영양제, 아이 스스로 챙겨요" |
| 24 | **전·후 이미지 대조** | 텍스트 B&A가 아닌 비주얼 대비 | 산만한 책상 vs 깔끔하게 집중하는 아이 |
| 25 | **다중 혜택 나열** | 한 제품의 여러 효과를 압축 나열 | "집중력 + 수면 + 감정 안정, 한 번에" |

## 5. 발작버튼 키워드 전략

**발작버튼**이란 타겟 페르소나가 보는 즉시 멈추는 감정적 트리거 단어.

### 브레인올로지 핵심 발작버튼
```
집중력, 산만함, 예민함, 짜증, 불안, ADHD
공부 못함, 학교 생활 걱정, 선생님 피드백
수면 문제, 잠 못 잠, 자다 깨다
```

### 카피 적용 방식
- 후킹 문구 첫 단어로 배치 (예: "산만한 아이 두신 엄마~")
- 부모의 내면 독백 형태로 자연스럽게 삽입
- 질문형과 결합 (예: "혹시 우리 아이 ADHD 아닐까 걱정되시나요?")

## 6. 타겟 피셔 문구 작성법

특정 페르소나를 콕 집어 부르는 문구. 광고 피로도를 낮추고 클릭률 극대화.

### 기본 공식
```
[자녀 나이] + [자녀 성별/상황] + [부모 역할/상태]
```

### 예시
- "7살 남아 키우는 엄마 주목"
- "초등 입학 앞둔 아이 부모님께"
- "담임선생님한테 '산만하다' 연락 받아보신 분"
- "오메가3 6개월 먹였는데 효과 없으신 분"

## 7. 스토리 유형별 카피 가이드

### A형 (정보형): 훅→문제→재정의→제품→성분→결과→CTA
- 후킹: 부정 카피 또는 비밀공개
- 본문: 재정의 기법 + 성분 과학화
- 톤: 전문적이지만 쉬운 언어

### B형 (공감형): 훅→공감→문제→제품→후기→CTA
- 후킹: 공감 상황 묘사 또는 타겟 피셔
- 본문: 친근한 문체 + 리뷰 강조
- 톤: 이웃 엄마처럼 편안하게

### C형 (B&A형): 훅→B&A→이유→제품→CTA
- 후킹: 변화 전후 대비 장면
- 본문: B&A 기법 + USP 강조
- 톤: 희망적, 긍정적

### D형 (데이터형): 훅→숫자/통계→문제→제품→근거→CTA
- 후킹: 숫자 강조
- 본문: 권위 인용 + 성분 과학화
- 톤: 신뢰감, 근거 중심

### E형 (역발상형): 훅→역발상→재정의→제품→CTA
- 후킹: 역발상 + 부정 카피
- 본문: 문제 재정의 기법
- 톤: 충격적, 의외성

### F형 (스토리형): 훅→스토리→감정고조→제품→CTA
- 후킹: 실패 경험 공유
- 본문: 스토리텔링 + 감정 고조
- 톤: 감성적, 내러티브

### G형 (비교형): 훅→비교→차별점→제품→CTA
- 후킹: 부정 카피 또는 질문형
- 본문: 경쟁 비교 재정의 + USP 강조
- 톤: 논리적, 비교 설득

### H형 (정보제공형): 훅→정보→팁→제품→CTA
- 후킹: 비밀공개 또는 권위 인용
- 본문: 가치 있는 정보 제공 후 자연스럽게 제품 연결
- 톤: 도움이 되는, 네이티브 광고 스타일

## 8. A/B 테스트용 변형 작성 가이드

후킹 문구 변형 3개를 항상 함께 작성:

- **변형 A**: 공감형 — 부모의 걱정·상황을 그대로 반영
- **변형 B**: 혜택형 — 직접적인 변화·이점 제시
- **변형 C**: 질문형 — 호기심·자기 진단 유도

### 예시
```
변형 A (공감형): "선생님한테 '산만하다' 연락 받으셨나요?"
변형 B (혜택형): "3주 만에 숙제 집중력 달라진 이유"
변형 C (질문형): "우리 아이 뇌에 뭐가 부족한 걸까요?"
```

## 9. 말 끝 다양화 유형

**규칙**: 스크립트 작성 시 반드시 아래 유형 중 하나를 적용. 한 스크립트 안에서 동일한 말끝이 반복되지 않도록 2~3가지 혼합 사용 권장.

| # | 유형 | 예시 |
|---|------|------|
| 1 | **~더라고 / ~더라구** | *"먹이고 나서 진짜 차분해지더라고"* |
| 2 | **~거든~ / ~거든요** | *"이게 카페인이 없거든요~ 그래서 먹였어"* |
| 3 | **~잖아 / ~잖아요** | *"애가 원래 젤리는 잘 먹잖아요"* |
| 4 | **~인 거 있지?** | *"근데 한 달 됐는데 진짜 달라진 거 있지?"* |
| 5 | **~이래 / ~이래요** | *"약국에서도 판대~ 몰랐지?"* |
| 6 | **~한다고~ (올려치기)** | *"이게 세로토닌 원료라고~?"* |
| 7 | **~맞지? / ~맞죠?** | *"나만 힘든 게 아니었던 거 맞죠?"* |
| 8 | **~지 않아? / ~지 않아요?** | *"이 정도면 한번 먹여볼 만하지 않아?"* |

| 10 | **~어떡해~ (공감)** | *"선생님한테 또 전화왔어요 어떡해~"* |
| 11 | **~하는 거 알았어?** | *"테아닌이 녹차에서 나오는 거 알았어?"* |

## 10. 감탄사·의성어 활용 유형

## 11. 설득 방식 유형

**규칙**: 모든 스크립트에 아래 유형 중 하나를 랜덤 적용. 한 스크립트 안에서 동일 방식 반복 금지.

### A. 논리형 (기존 방식)

| 유형 | 구조 | 예시 |
|------|------|------|
| **악순환 논리** | 스트레스→예민→더 스트레스. 고리를 끊어야 함 | *"예민할수록 더 예민해지는 악순환, 고리를 끊어야 해요."* |
| **원료 논리** | 몸이 못 만드는 걸 직접 넣어줘야 함 | *"세로토닌은 트립토판 없이 못 만들어요. 트립토판은 먹어야만 생겨요."* |
| **접근 오류 논리** | 좋은 걸 먹었는데 안 됐다 → 방향이 틀렸기 때문 | *"열심히 먹였는데 효과 없었다면, 아이한테 필요한 성분이 그게 아닌 거예요."* |
| **호르몬 메커니즘** | 스트레스→코르티솔↑→세로토닌·도파민 불균형→산만·예민 | *"코르티솔이 올라간 상태에서는, 아이도 참고 싶어요. 근데 몸이 말을 안 들어요."* |
| **타이밍 논리** | 기질은 어릴 때 잡아야. 굳어지면 더 어려움 | *"초등 저학년이 골든타임이에요. 이때 패턴이 잡히면 훨씬 수월해져요."* |
| **리스크 제거 논리** | 불확실하면 안 사는 게 당연 → 불안 먼저 제거 | *"14일 먹여보고, 아니면 전액 돌려드려요."* |

### B. 감성·수사형 (신규)

| 유형 | 구조 | 예시 |
|------|------|------|
| **비유형** | 논리를 이미지로 치환 | *"배고픈 아이한테 물만 먹이는 거예요. 원인에 맞는 성분이어야 해요."* |
| **반문 후 침묵형** | 질문만 던지고 0.5초 공백, 보는 사람이 생각하게 | *"근데 생각해봐요. 왜 열심히 먹였는데 안 됐을까." (공백) "성분이 달랐던 거예요."* |
| **고백 후 반전형** | "처음엔 ㅋㅋ했거든" 시작해서 뒤집기 | *"영양제로 기질이 바뀐다는 거 처음엔 ㅋㅋ했거든. 근데 지금은 재구매 4번째야."* |
| **공감 폭발형** | 논리 없이 감정만으로 설득 | *"내가 잘못 키운 건지, 아이가 원래 이런 건지. 근데 둘 다 아니었어."* |
| **숫자 던지고 떠나기** | 설명 없이 숫자가 말하게 | *"200만 개. 그게 다예요."* |
| **상상 유도형** | "만약에" 구조 | *"만약에 아이가 숙제 앉아서 끝낸다면. 그게 가능한 일이에요."* |
| **타인 시각 활용형** | 제3자 반응을 보여줌 | *"처음엔 '무슨 영양제가 기질을 바꿔'라고 했어요. 지금은 본인이 주문해요."* |
| **호르몬 감정이입형** | 메커니즘을 아이 입장으로 번역 | *"코르티솔이 올라간 상태에선 아이도 참고 싶어요. 근데 몸이 말을 안 들어요. 의지 문제가 아니에요."* |
| **역설형** | 뻔한 결론을 뒤집기 | *"차분한 아이로 키우고 싶다면서, 차분하게 만드는 성분은 안 챙겨줬던 거잖아요."* |
| **선택지 제거형** | 선택지를 좁혀서 결론으로 유도 | *"약은 싫고, 그냥 두기엔 불안하고. 그 사이에 있는 게 이거예요."* |

## 13. 성분 설명 표현 다양화

**규칙**: 성분 설명이 등장하는 씬에서 매번 `"테아닌 150mg"` 형식만 사용하면 스크립트가 획일적으로 보임. 아래 유형 중 상황에 맞게 선택하여 사용. 한 캠페인 내 복수 스크립트 작성 시 동일 유형 반복 금지.

| 유형 | 구조 | 예시 |
|------|------|------|
| **함량형** | 성분명 + mg | `"테아닌 150mg, 트립토판 150mg"` |
| **기능 중심형** | 기능 설명 + 성분명 | `"스트레스 낮춰주는 테아닌"` / `"긴장 풀어주는 성분, 테아닌"` |
| **메커니즘형** | 작동 원리 + 성분명 | `"코르티솔 줄이는 핵심 성분인 테아닌"` / `"흥분한 신경을 진정시키는 가바"` |
| **호르몬 연결형** | 호르몬 관계 설명 | `"세로토닌을 만드는 재료, 트립토판"` / `"뇌 안정 호르몬의 원료인 트립토판"` |
| **결과 중심형** | 아이에게 나타나는 변화 | `"예민함을 잡아주는 가바"` / `"산만함을 줄여주는 테아닌"` |
| **감성 번역형** | 성분 기능을 소비자 언어로 | `"아이 뇌가 스스로 조용해지는 힘, 테아닌"` |
| **함량+기능 조합형** | 함량 언급 후 기능 추가 | `"테아닌 150mg — 긴장완화에 도움을 주는 성분"` |

### 적용 원칙
- 같은 캠페인 내 스크립트 5개를 만들 경우, 5개 모두 다른 유형 사용
- 한 스크립트 안에서 성분 3종(테아닌·트립토판·가바)을 나열할 때 각각 다른 유형 혼합 가능
- 컴플라이언스 준수: 어떤 유형이든 `테아닌="긴장완화에 도움"`, `트립토판="세로토닌 원료"` 범위 내에서 표현

## 14. 스크립트 배포 전 필수 검수 프로세스

스크립트 작성 완료 후 배포 전, 반드시 아래 항목을 직접 검수하고 수정한 뒤 배포한다.

### 검수 항목

| # | 항목 | 체크 포인트 |
|---|------|------------|
| 1 | **화자 일관성** | 스크립트 화자(엄마/아이/나레이터)가 처음부터 끝까지 동일한가 |
| 2 | **주어-서술어 매칭** | "스스로 먹이더라"처럼 주체가 불분명한 서술 없는가 |
| 3 | **말투 일관성** | 존댓말/반말이 중간에 섞이지 않는가. 기본 말투 베이스 유지 |
| 4 | **감탄사/표현 반복** | 동일 감탄사(근데 있잖아, 어머~ 등)가 한 스크립트에서 2회 이상 반복되지 않는가 |
| 5 | **화자 나이·성별 적합성** | 아이 목소리 스크립트에 어른 감탄사(어머~, 글쎄! 등) 사용되지 않는가 |
| 6 | **마무리 문장 완결성** | 감성 문장으로 끝나는 경우 자연스럽게 매듭짓는 마무리 문장이 있는가 |
| 7 | **브랜드 vs 화자 관점** | 마무리 멘트가 브랜드 말투("~함께할게요")가 아닌 화자(엄마/아이) 입장으로 쓰였는가 |
| 8 | **컴플라이언스** | 금지 표현(의약품 효능 암시, 오메가3 비교) 없는가 (ADHD 등 증상 키워드는 자유롭게 사용 가능) |
| 9 | **말끝 태도 모순** | 같은 사실을 ~잖아요(이미 앎) + ~이래요(새 정보)로 연이어 쓰지 않았는가 (→ Section 15 참고) |
| 10 | **공감형↔정보형 충돌** | ~지 않아요?(공감 질문) 바로 뒤에 ~이래요(정보 전달)가 오지 않는가. 인과 흐름은 ~기 때문에/~는데로 연결 (→ Section 15 참고) |

| 9 | **말끝 태도 모순** | 같은 사실을 ~잖아요(이미 앎) + ~이래요(새 정보)로 연이어 쓰지 않았는가 (→ Section 15 참고) |
| 10 | **공감형↔정보형 충돌** | ~지 않아요?(공감 질문) 바로 뒤에 ~이래요(정보 전달)가 오지 않는가 (→ Section 15 참고) |
| 11 | **화자 관점 명확성** | 화자 본인의 과거 생각/감정은 `~인 줄 알았거든요/싶었거든요`. `~더라고요`는 타인 관찰 또는 새로운 발견에 사용. 혼용하면 화자가 자신을 3인칭으로 서술하는 것처럼 들림 |
| 12 | **단어 정확성** | 사실·정보 전달 문장에 `이야기`, `부분` 같은 추상·모호한 단어 사용 금지. `문제`, `원인`, `이유` 등 명확한 단어 사용 |
| 13 | **결론-전제 일치** | 감성 결론 문장의 전제가 앞 씬에서 충분히 설정되었는가. 전제 없는 결론은 공허함. 예: "엄마 탓이 아니었던 거잖아요" → 앞에서 엄마가 자책하는 장면이 반드시 있어야 함 |

### 검수 실패 시
수정 완료 후 배포. 배포 전 검수를 생략하지 않는다.

오메가3와 뉴턴젤리는 교차 선택하는 동일 카테고리 제품이 아님. 브레인올로지에는 오메가3 제품도 있으며, 뉴턴젤리 스크립트에서 오메가3를 경쟁 상대로 언급하면 자사 제품 간 충돌이 발생함. 일반 영양제와의 비교가 필요한 경우 "기존에 먹이던 영양제", "다른 영양제" 등 카테고리 수준으로만 표현할 것.

## 15. 문장 말끝 논리 일관성 규칙

스크립트에서 말끝 어미는 단순한 말투 선택이 아니라 **화자의 인식 태도**를 결정한다. 한 문장 안에서 또는 연속된 두 문장에서 태도가 충돌하면 전달력이 떨어지고 어색해진다.

### 패턴 1 — ~잖아요 + ~이래요 충돌 금지

| 상황 | 문제 예시 | 이유 | 수정 방향 |
|------|----------|------|----------|
| 같은 사실에 대해 "알고 있다" + "처음 들었다" 동시 표현 | `"세로토닌이 무너진다는 거잖아요. 그게 진짜 원인이래요."` | ~잖아요=이미 알고 있음을 전제, ~이래요=새 정보처럼 전달 → 동일 정보를 두 가지 태도로 이중 서술 | 첫 문장을 연결어로 바꿈: `"세로토닌이 무너지는데, 그게 진짜 원인이래요."` |

### 패턴 2 — 공감 질문 + 정보 전달 충돌 금지

| 상황 | 문제 예시 | 이유 | 수정 방향 |
|------|----------|------|----------|
| 공감을 구한 직후 새 정보를 줌 | `"더 산만해지는 거지 않아요? 고리를 끊어줘야 하는 거래요."` | ~지 않아요?=시청자 공감 유도(이미 알고 있죠?), ~이래요=화자가 새로 알려주는 정보 → 공감을 구해놓고 바로 정답을 주면 흐름 단절 | 인과 접속으로 통일: `"더 산만해지기 때문에, 고리를 끊어줘야 하는 거래요."` |

### 인과관계 표현 원칙

정보 전달 씬(나레이터·전문형 VO)에서 원인→결과 관계는 반드시 논리 접속어로 연결한다.

| 용도 | 올바른 표현 | 잘못된 표현 |
|------|-----------|-----------|
| 원인→결과 연결 | `~기 때문에`, `~는데`, `~어서`, `~으니까` | `~잖아요? + ~이래요` |
| 결론 도출 | `그러니까 ~이래요`, `그래서 ~거래요` | `~지 않아요? + ~이래요` |
| 공감 후 마무리 | 공감 질문이면 공감으로 끝내거나 `~잖아요`로 통일 | 공감 질문 후 정보형 어미로 전환 |
