## A. 말투·문장구조·설득흐름 [전체 37개 기반]

### A-1. 검증된 훅(Hook) 패턴

전체 영상에서 반복 검증된 첫 문장 유형은 6가지다.

**① 증상·상황 직접 호명형** — 타겟이 자신의 이야기임을 바로 인지하게 만드는 첫 문장
- "진짜 한순간도 가만히 있지 않는" → 아이 행동 묘사로 시작
- "밥만 먹으면 배가 아프다는 아이" → 상황 서술로 훅
- "울보 아이들 키우는 엄마들" → 타겟 직접 호명
- "요새 자꾸 잘 시간만 되면 배아프다고 오는데" → 일상 상황 묘사
- "공부하다 말고 자리이탈?" → 짧은 행동 묘사 질문형
- "조금만 마음에 안 들어도 편식하는 아이?" → 질문형 행동 묘사
- "아이 오메가3 좋다는 건 알겠는데" → 소비자 마음 대변형

**② 숫자·등급 충격형** — 구체적 수치나 등급으로 신뢰와 주목을 동시에
- "아이 면역력의 70%는 장에서 시작"
- "유산균보다 100배 강한 살균력"
- "유해균의 95%는 대장에 몰려"
- "한포에 455억 마리를 넣었습니다"
- "3일만 먹어도 5시간 뛰어논 것처럼 차분해지고"
- "EPA+DHA 980mg, 98% 고순도"
- "2주 정도 먹이니까 편식이 줄기 시작하더니"

**③ 역발상·금지형** — "하면 안 된다", "틀린 방법" 등으로 역설적 주목 유도
- "유산균 먹지 마세요!"
- "이건 틀린 방법이에요!"
- "이런 오메가3 먹으면 큰일 난다"
- "7년차 약사가 절대 안 먹이는 아이 오메가3"
- "이거 방치하면 절대 안 되는데요"

**④ 의외의 원인 제시형** — "예민해서 그런 게 아니라 원인은 ~"
- "산만함의 원인이 스트레스라는 점 알고 계셨나요?"
- "그저 입맛이 까다롭다고만 생각했는데, 사실 HSP라고요?"
- "예민해서 그런가 했는데 원인은 심리적 스트레스"
- "사소한 일에도 우는 아이, 그거 예민해서 그런 게 아니라 원인이 과각성일 수 있습니다"

**⑤ 고백·실패 경험 공유형** — "나도 그랬다" 방식으로 공감부터 획득
- "솔직히 말하면 이게 무슨 예민함을 바꿔주나 싶었거든요"
- "사실 아이들 오메가3 비려서 잘 안 먹잖아요. 그래서 반신반의하면서 줬는데"
- "아무리 유명하고 좋은 유산균 다 먹여봐도 아이가 거부하면 소용 없겠죠?"
- "유명한 거 다 사봤거든요. 퉤..."

**⑥ 체크리스트·목록형** — 숫자 나열로 즉각적 자기진단 유도
- "1. 아이 배가 빵빵하다 / 2. 설사나 변비가 반복된다"
- "1. 다크서클이 있다 / 2. 맑은 콧물이 잦다"
- "Q. 아이가 감정 기복이 크고 예민해요 / Q. 작은 일에도 쉽게 상처받고 울어요"
- "아랫배 가스, 변 닦을 때 묻어나옴 — 이런 사람은 유산균 먹지 마세요"

### A-2. 나레이션 전개 구조 & 감정선

37개 영상을 통틀어 공통적으로 확인되는 감정선 진행 패턴은 다음과 같다:

**패턴 1 — 공감 → 원인 재정의 → 해결책 → 제품 → CTA** (가장 빈번)
```
[공감] "밥만 먹으면 배가 아프다는 아이. 저희 아이가 몇 달 동안 그랬거든요."
[원인] "처음엔 먹기 싫어서 그런가 했는데. 장 내 환경이 무너지고 있는 상태."
[해결] "유익균은 늘리고 유해균은 줄여 악순환을 끊어줘야 하는데요."
[제품] "그래서 저는 딱 3가지를 기준으로 골랐어요."
[CTA] "아래 링크 참고해주세요!"
```
적용 영상: 고생, 도윤맘, 편식, 울보아들, AD전화 등

**패턴 2 — 충격 훅 → 메커니즘 설명 → 기존 해결책 비판 → 올바른 해결책 → 제품 → CTA** (정보형)
```
[충격 훅] "유해균의 99%는 대장에 몰려 있어요."
[메커니즘] "얘네가 음식물을 부패시켜 독소를 만들어요."
[기존 비판] "그래서 유산균 많이 먹이시는데, 이건 틀린 방법이에요!"
[올바른 해결] "유산균은 소장에서만 살고 대장에서는 죽어버리거든요."
[제품] "그래서 비피더스균이에요."
[CTA] "아래 링크 클릭해보세요!"
```
적용 영상: 비피젠 시리즈, 단약사, 필약사 등

**패턴 3 — 스토리 → 전환점 → 변화 → 현재 → CTA** (오가닉·B&A형)
```
[스토리] "ADHD란 말을 처음 들은 건 첫째가 학원을 옮기던 날이었어요."
[전환점] "소아과 의사분께 조언을 구했어요. 아침 등교 전 보조제를 먹였고요."
[변화] "시간이 좀 지나니까... 집중력이 눈에 띄게 좋아졌다고 칭찬을 하시더라고요."
[현재] "지금도 둘째도 먹이기 시작했어요."
[CTA] 생략 or "자연유래라 부작용 없어서 계속 먹일 것 같아요."
```
적용 영상: AD전화, 정이엘, 예민꿀템, 남편 등

**패턴 4 — 제품 소개 → 특징 나열 → 경험담 → CTA** (커머셜·베네핏형)
```
[제품 소개] "약을 안 먹어도 산만함을 감소시켜주는 뉴턴젤리인데요."
[특징 나열] "산만·예민 기질에 필요한 3가지 성분. 현직 약사 추천템."
[경험담] "영양제 거부하던 아이도 직접 찾아 먹어요."
[CTA] "할인기간 꼭 끝나니까 이 기회 놓치지 마세요."
```
적용 영상: 속삭임, 남편, 비피젠_유명인필수 등

### A-3. 자막 문체 & 어미 패턴

**검증된 어미 조합 (가장 많이 등장한 순서)**

| 어미 유형 | 실제 등장 문장 예시 | 등장 맥락 |
|---|---|---|
| ~더라고요 / ~더라구요 | "먹이고 나서 진짜 달라지더라고요", "젤리처럼 맛있다고 잘 먹더라고요" | 화자의 새로운 발견·경험 전달 |
| ~거든요 / ~거든요? | "이건 약국에도 입점되어서 믿고 먹일 수 있겠더라고요", "현직 약사 추천템으로도 자주 등장하거든요?" | 부연 설명·이유 제시 |
| ~잖아요 | "아이들 오메가3 비려서 잘 안 먹잖아요", "거부하면 소용 없겠죠?" | 이미 알고 있는 사실에 동의 구하기 |
| ~래요 / ~이래요 | "맛있다고 먼저 찾는 제품이에요", "아이 스스로 느낀다네요" | 새로운 정보 전달 |
| ~어요 / ~이에요 | "원인은 과각성일 수 있습니다", "성장기에는 장 건강을 챙겨주는 게 가장 중요해요" | 정보·사실 전달 |
| ~는데요 / ~는데 | "이 신경계를 안정 상태로 만들어서 잘 넘어갈 줄 아는 아이로 키우는 방법 알려드릴게요" | 전환 연결어 |
| ~고 있는데 / ~고 있어요 | "요즘 저녁마다 이걸 챙겨주고 있는데", "지금도 첫째는 꾸준히 먹이고 있고요" | 현재 진행 습관화 표현 |
| ~었어요 / ~았어요 | "드디어 정착했어요!", "집중력이 달라지더라구요" | 변화 결과 고백 |

**자막 호흡 단위 (flowing 기준)**
- 한 자막당 5~15자 이내가 기본
- 감정 강조 포인트에서는 3~5자 단문으로 끊음 ("퉤...", "완전 나인데", "드디어 정착했어요!")
- 근거/설명 문장은 15~25자로 한 줄 처리
- 체크리스트는 한 항목 = 한 자막으로 분리

**자막 강조 부호 패턴**
- 느낌표(!) — 결론·혜택·CTA 문장에 필수
- 물음표(?) — 훅·공감 질문에 사용
- 말줄임표(...) — 고백·감성 전환 직전
- 체크마크(✓) — 제품 특장점 나열 시

### A-4. 영상 유형별 구조 특징

**오가닉형 자막 구조**
- flowing 자막이 주도, fixed는 브랜드명 or 핵심 기능성 문구 1~2개
- 나레이션과 자막이 거의 일치하되, 자막은 나레이션보다 2~3단어 더 간결
- 개인 경험담 어조: "저희 아이가", "저는", "엄마 입장에서"
- 일상 UGC 장면에 자막 오버레이 방식

**정보형 자막 구조**
- fixed 자막으로 핵심 수치·성분명·기능성 키워드 고정 노출
- flowing은 메커니즘 설명을 호흡 단위로 분절
- 의학 용어 → 소비자 언어 번역 자막 패턴 (예: "대장 유해균 → 독소 → 전신 염증")
- 숫자 등장 시 단독 자막으로 강조 ("95%", "100배", "455억 마리")

**커머셜형 자막 구조**
- fixed 자막에 가격 정보·할인율·CTA 버튼 고정
- flowing에 소셜프루프("맘카페 난리", "약국 품절", "1500만 정 돌파") 포함
- 긴박감 표현 어미: "오늘까지", "선착순", "곧 품절"

**베네핏형 자막 구조**
- 혜택 항목을 체크리스트 또는 나열 구조로 배치
- "✓ 비린 맛 거의 없고 / ✓ 씹어먹는 츄어블" 같은 짧은 포맷
- fixed 자막에 환불 보장, 인증 정보 고정

## B. 제품별 키워드·워딩·논리 [제품명 있는 영상 기반, 그대로 써도 OK]

> 이 섹션의 모든 문장과 표현은 실제 레퍼런스 영상에서 추출된 것으로, 스크립트 작성 시 그대로 사용 가능하다.

### B-1. 뉴턴젤리

**분석 대상 영상 (자사):** 상상, 울보아들, 도윤맘, 예민꿀템, 남편, 드세요, AD전화, 편식, 속삭임

#### 핵심 키워드 & 표현

**증상 키워드 (타겟이 즉각 공감하는 단어)**
- 산만함, 예민함, 집중력 저하, 짜증, 감정 기복
- 자리이탈, 공부 거부, 뒤척임, 과각성
- HSP, ADHD 직전, 과각성 신경계
- 스트레스, 불안감, 과민반응, 긴장

**성분 키워드 (식약처 인증 기반)**
- 테아닌 — "긴장완화에 도움", "아이의 긴장감과 예민함을 낮춰줘요", "스트레스를 낮춰주는 루틴"
- 가바 — "작은 일에도 쉽게 상처받고 우는 아이", "흥분 신경 진정"
- 트립토판 — "잠드는 데 오래 걸리고 밤새 뒹굴뒹굴"
- 글리신 — "밤이 되면 오히려 더 예민해져요"
- 비타민B — "평소 긴장을 많이 하고 쉽게 지쳐요"

**제품 특성 키워드**
- 샤인머스캣 맛 / 달달한 샤인머스캣 맛
- 아이가 먼저 찾는 영양제
- 간식처럼 1포씩
- 약국에도 입점 / 현직 약사 추천템
- 자연유래 / 부작용 없어서
- 젤리형 / 스틱 포
- 브레인올로지 L3 뉴턴젤리

#### 설득 논리 & 문제제기 방식

**핵심 재정의 논리 1 — "예민한 게 아니라 스트레스가 원인"**
```
표면 문제: 산만하다, 예민하다, 짜증낸다, 자리이탈
↓ 재정의
근본 원인: 심리적 스트레스 / 해소 방법을 모르는 아이
↓ 메커니즘
스트레스 → 도파민 과다 분비 → 수면장애·주의력·집중력 저하 → 불안감
↓ 솔루션
스트레스를 낮춰주는 루틴 (테아닌 성분 = 긴장완화)
```
실제 나레이션: "원인은 심리적 스트레스. 해소 방법을 모르는 아이는 산만하고 예민해지는데, 아이한테 필요한 건 혼내는 훈육이 아니라 스트레스를 낮춰주는 루틴이었어요."

**핵심 재정의 논리 2 — "과각성 신경계"**
```
표면 문제: 사소한 일에도 운다
↓ 재정의
원인: 과각성 (신경계가 늘 긴장 상태)
↓ 메커니즘
작은 자극 → 큰 위협으로 인식 → 감정 순간 증폭
↓ 솔루션
신경계를 안정 상태로 만드는 성분
```
실제 나레이션: "그거 예민해서 그런 게 아니라 원인이 과각성일 수 있습니다. 신경계가 늘 긴장 상태라 작은 자극도 큰 위협으로 느껴서 감정이 순간적으로 증폭되는데요."

**핵심 재정의 논리 3 — "HSP 기질"**
```
표면 문제: 편식, 사회성 문제
↓ 재정의
원인: HSP (선천적으로 감각에 높은 민감성을 가진 기질적 특성)
↓ 긍정적 전환
잘 해결하면: 섬세하고 예술적인 감수성 → 예술계 인재
↓ 솔루션
불안감 완화·차분한 안정 성분 (테아닌)
```
실제 나레이션: "이걸 해결하기만 하면 HSP 특유의 섬세하고 예술적인 감수성이 살아나 예술계에 한 획을 긋는 인재가 될 수 있을 텐데."

**핵심 재정의 논리 4 — "훈육이 아니라 보조영양"**
```
부모의 시도: 태권도, 요가, 행동 교정, 훈육
```

## C. 콘텐츠 유형별 패턴 [즉시 적용 가이드]

### C-1. 오가닉형

#### 세부 유형 & 각 구조

**세부 유형 A — "엄마 고백형"** (가장 기본)
- 구조: 일상 상황 묘사 → 아이의 문제 행동 → 해결책 발견 → 제품 경험 → 현재 상태
- 화자: 엄마 1인칭 ("저희 아이가", "저는", "제가")
- 어조: 이웃 엄마에게 말하듯 편안하게
- 예시 영상: 고생, 도윤맘, 퉤, 찾아먹어, 편식

**세부 유형 B — "아이 반응 중심형"**
- 구조: 기존 실패 경험 → 이 제품은 다름 → 아이 반응 (먼저 찾아 먹는다) → 편안해진 엄마
- 핵심 포인트: 아이가 "스스로 찾아 먹는다"는 장면 강조
- 예시 영상: 찾아먹어, 잘시간, 김유니, 새학기

**세부 유형 C — "정보+공감 혼합형"**
- 구조: 발견한 새 정보 (원인 재정의) → 그래서 제품 선택 → 효과 고백
- 예시 영상: 울보아들, 편식, AD전화

#### 검증된 훅 문장 예시
- "밥만 먹으면 배가 아프다는 아이"

- "요새 자꾸 잘 시간만 되면 배아프다고 우는데"
- "공부하다 말고 자리이탈?"

#### 어미 & 말투 특징
- ~거든요 (부연 설명), ~더라고요 (새 발견), ~잖아요 (공감 확인)
- ~었어요 (결과 고백), ~려고요 (의지 표명)

### C-3. 정보형

#### 세부 유형 & 각 구조

**세부 유형 A — "원인 재정의형"** (가장 많음)

- 예시 영상: 비피젠 시리즈, 상상, 단약사

**세부 유형 B — "전문가 추천형"**
- 구조: 전문가 신뢰 세팅 → 기준 제시 → 제품이 기준 충족 → CTA
- 특징: 약사·영양사·의사 권위 활용, fixed 자막에 자격증·경력 명시
- 예시 영상: 단약사, 필약사, 김제연약사님, 등급

**세부 유형 C — "Q&A·체크리스트형"**
- 구조: 증상 질문 → 성분 답변 → 한 제품으로 모두 해결 → CTA
- 특징: flowing 자막이 질문-답변 쌍으로 구성
- 예시 영상: 드세요

#### 정보 전달 방식 패턴
- 숫자 강조: "70%", "95%", "100배", "455억 마리" — 단독 자막으로 임팩트 강조

- 메커니즘 가시화: 장기 해부도·3D 의료 일러스트·인포그래픽 활용
- 단계별 전개: "첫째... 둘째... 셋째..." 또는 번호 목록 사용

#### 인트로 (0-5초) — 86개 visual

공통 목적: 스크롤 멈춤. 타겟이 자신의 상황임을 0.5초 안에 인지하게 만들기.

**가장 많이 쓰인 인트로 소스 유형 (빈도순)**
1. 아이 증상·일상 행동 장면 — 바닥 앉기·신발 신기·태블릿 보기·엄마와 놀기 (극히 일상적인 장면으로 시작)
2. 아이 불편한 증상 클로즈업 — 배 만지는 모습·멍한 표정·눈 비비는 모습·울고 있는 표정
3. 성인 인물 얼굴 클로즈업 — 엄마·약사가 직접 카메라 응시, 자기 고백 시작 (고백 또는 금지 훅)
4. 텍스트 카드 단독 — 검정 배경·흰 텍스트로 충격 팩트 첫 등장 (정보형에서 주로 사용)
5. 3D 캐릭터·의료 일러스트 — 유아 복부 팽만·대장 의료 영상 (타사 정보형에서 주로 사용)
6. 제품 직접 등장 — 패키지 정면 또는 언박싱 시작 (커머셜에서만 인트로에 제품 사용)

**인트로 금지 패턴**: 제품 먼저 등장 (오가닉·정보형에서는 비효과적), 밝고 행복한 아이 장면으로 시작 (공감 유발 실패)

---

#### 미들 (6-25초) — 273개 visual

공통 목적: 문제 심화 → 원인 설명 → 해결책 제시. 인포그래픽·설명 구간.

**오가닉 미들 소스**
- 아이 행동 장면 (위에서 내려다보는 앵글, POV 유지)
- 제품 클로즈업 + 아이 일상 병치 (엄마 손에 들린 제품 패키지 전면, 배경에 아이 뛰어노는 모습)
- 인포그래픽 카드 (건강한 장 vs 무너진 장 비교 등 간단한 정보만)

**정보형 미들 소스**
- 연구논문 스크린샷 / 데이터 테이블 (P-value 강조, 그래프)
- 의료 3D 애니메이션 (소화기관·뇌·박테리아 전투 시뮬레이션)
- 엄마·약사 상반신 설명 (손가락 포인팅·제스처)
- 아이 행동 장면 (문제 상황 지속 묘사)

**커머셜 미들 소스**
- 뇌 3D 그래픽 (파란 발광·신경망·전두엽 활성화 시각화)
- 제조공정 영상 (기계 라인·품질검사·연구자 현미경)
- 성분 인포그래픽 (FDA 균주·5in1·98% 순도 등 스펙 강조)

---

#### 아웃트로 (25초+) — 231개 visual

공통 목적: 변화 증거 제시 → 제품 소개 → CTA (검색/구매).

**가장 많이 쓰인 아웃트로 소스 유형 (빈도순)**
1. 아이 제품 섭취·체험 장면 — 스스로 포장 뜯기·엄마가 건네주기·입에 넣는 순간
2. 아이 만족·변화 장면 — 환하게 웃기·엄지 표시·손가락 가리키며 웃기·뛰어오는 블러 컷
3. 제품 클로즈업 — 손으로 들고 카메라 향해 내밀기·성분표 가리키기·박스+낱개 함께 배치
4. 인물 고백 마무리 — 엄마·약사가 카메라 응시하며 추천 마무리 멘트
5. 상품 정보 인포그래픽 — 가격·할인율·배송 정보 (커머셜만)
6. 검색창·네이버 카페 게시글 — 초록창 검색 유도, 실제 후기 스크린샷 (오가닉·정보형)

```

```

```
제품 박스 클로즈업 (손으로 측면 체크포인트 짚으며 보여주는 UGC 톤)
```

## E. 크로스 인사이트: 전체 영상 통합 패턴

### E-1. 가장 많이 쓰인 설득 구조 TOP 3
1. **역발상 + 재정의**: "~인 줄 알았는데, 사실 원인은 ~이에요" — 기존 상식 뒤집기
2. **방치 공포 + 해결**: "이걸 방치하면 ~가 됩니다. 그래서 ~를 써야 해요" — 공포 자극 후 안도 제공
3. **실패 경험 + B&A**: "다 사봤는데 안 먹었어요. 근데 이건~" — 공감 획득 후 차별화

### E-2. 검증된 자막 전환 패턴
- "그런데 / 근데" — 반전·전환 직전에 배치 (가장 빈번)
- "사실은 / 알고 보니" — 재정의 직전
- "이게 정말 효과가 있을까?" → 짧은 공백 → 변화 결과 — 의심 후 반전
- "가장 신기한 건" — 추가 베네핏 reveal 시
- "그래서" — 원인 → 해결책 연결

### E-3. 오가닉 vs 커머셜 구분 실전 기준

| 항목 | 오가닉 | 커머셜 |
|---|---|---|
| 화자 | 엄마·사용자 | 브랜드 또는 사용자+브랜드 |
| 가격 자막 | 없음 | 반드시 포함 |
| CTA | 소프트 ("검색해보세요") | 하드 ("지금 바로 구매") |
| 감정선 | 공감·발견·변화 | 긴박감·희소성·혜택 |
| 스틱 포 | 상품명 자연스럽게 | 상품명+가격+할인율 강조 |

## F. 스크립트 다양성 강제 규칙 [스크립터 에이전트 필수 준수]

> 이 섹션은 같은 캠페인 내 스크립트들이 구조·흐름·표현이 반복되지 않도록 강제하는 규칙이다.
> 스크립트 작성 전 반드시 이 섹션을 읽고, 현재 배치에서 사용된 패턴을 추적한 후 다음 스크립트를 결정해야 한다.

---

### F-1. 훅 유형 로테이션 테이블

같은 캠페인 내 5개 스크립트 작성 시, 아래 6가지 훅 유형 중 최소 4가지를 각각 다른 스크립트에 사용해야 한다.

| ID | 훅 유형 | 핵심 공식 | 사용 예시 |
|---|---|---|---|
| H1 | 증상·상황 직접 호명형 | "~하는 아이 / ~하는 엄마" | "공부하다 말고 자리이탈?" |
| H2 | 숫자·등급 충격형 | 구체 수치로 시작 | "200만 개 넘게 팔렸어요" |
| H3 | 역발상·금지형 | "하면 안 돼" / "틀린 방법" | "이거 방치하면 절대 안 되는데요" |
| H4 | 의외의 원인 제시형 | "~인 줄 알았는데 사실 원인은 ~" | "예민한 게 아니에요, 과각성이에요" |
| H5 | 고백·실패 경험 공유형 | "나도 그랬어" 공감 획득 | "저도 처음엔 반신반의했거든요" |
| H6 | 체크리스트·목록형 | 증상 나열로 자기진단 유도 | "Q. 조금만 지나면 딴짓해요 / Q. 잠들기까지 한 시간" |

**5개 배치에서 동일 훅 유형 2회 사용 금지.**

---

### F-2. 전개 구조 로테이션 테이블

| ID | 전개 패턴 | 요약 | 최대 사용 횟수(5개 배치) |
|---|---|---|---|
| P1 | 공감 → 원인 재정의 → 해결 → 제품 → CTA | 오가닉·공감형 기본 | 2회 (연속 금지) |
| P2 | 충격 훅 → 메커니즘 → 기존 비판 → 해결 → 제품 → CTA | 정보형 기본 | 2회 (연속 금지) |
| P3 | 스토리 → 전환점 → 변화 → 현재 → CTA | 오가닉 B&A형 | 1회 |
| P4 | 제품 소개 → 특징 나열 → 경험담 → CTA | 커머셜 기본 | 1회 |

**같은 캠페인 내 P1이 3개 이상이면 즉시 1개를 P3 또는 P4로 교체한다.**

### F-3. 콘텐츠 유형 배분 원칙

**5개 배치 권장 구성 예시**

| 스크립트 번호 | 콘텐츠 유형 | 세부 유형 | 훅 유형 | 전개 패턴 |
|---|---|---|---|---|
| 1 | 오가닉 | 엄마 고백형 | H5 | P1 |
| 2 | 정보형 | 원인 재정의형 또는 Q&A형 | H4 또는 H6 | P2 |
| 3 | 오가닉 | 아이 반응 중심형 또는 가족 갈등+반전형 | H1 | P3 |
| 4 | 커머셜 | 후기 기반 전환형 | H2 또는 H5 | P4 |
| 5 | 정보형 또는 베네핏형 | 전문가 추천형 또는 복합 기능형 | H3 또는 H2 | P2 또는 P1 |

이 배분은 참고 예시이며, 브리프 콘텐츠에 맞게 조정 가능. 단, 아래 **절대 금지 패턴**은 준수.

---

### F-4. 절대 금지 반복 패턴 (같은 캠페인 내 2개 이상 존재 금지)

1. **"선생님 연락" 구조**: 선생님에게 문제 연락 → 아이 변화 후 칭찬 연락 반전
2. **"맘카페 발견" 구조**: 맘카페에서 봄 → 반신반의 → 효과 확인
3. **"코르티솔 메커니즘" 설명**: 스트레스→코르티솔↑→세로토닌/도파민 불균형 동일 흐름도
4. **"성분 3종 순서 나열"**: 테아닌 → 트립토판 → 가바 동일 순서로 소개
5. **"아이도 엄마도 편안해요"** 마무리 문장 그대로
6. **"8BOX 149,900원 + 환불 보장"** CTA 구조 그대로
7. **엄마 얼굴→제품→엄마+아이 투샷** 3장면 순서 구조
8. **"남편이 회의적이었다" 도입** → 남편이 먼저 재주문한 반전 구조
9. **"약은 부작용이 걱정되고"** → 자연유래 성분으로 연결하는 불안 해소 구조
10. **훅~마무리 VO 50%이상** 이전 스크립트 문장과 동일하거나 단어 치환 수준의 유사

### F-5. 화자·설정 다양성 원칙

5개 스크립트 내에서 아래 화자 중 최소 2가지를 사용한다:

| 화자 유형 | 특징 | 예시 콘텐츠 유형 |
|---|---|---|
| 엄마 1인칭 고백형 | "저희 아이가", "저는" | 오가닉 |
| 나레이터형 | 무화자 정보 전달 | 정보형 |
| Q&A 진행자형 | "Q. ~~ A. ~~" | 정보형 체크리스트 |
| 약사·전문가형 | "현직 약사로서" | 커머셜·정보형 |
| 아이 시점 묘사형 | 아이 행동·표정 장면 중심 | 오가닉 |

### F-6. 시각 구성 다양성 원칙

| 구성 | 설명 |
|---|---|
| 인물 클로즈업 중심 | 엄마 얼굴 직접 응시 장면 다수 |
| 텍스트 카드 중심 | 검정 배경 텍스트만으로 메시지 전달 |

| 아이 일상 UGC 중심 | 탑뷰·POV·일상 장면 다수 |
| 제품 클로즈업 중심 | 제품 정물·손에 든 제품 중심 |

5개 중 동일 구성 3개 이상이면, 나머지는 다른 구성으로 변경한다.

## G. 초반 0~5초 훅 구조 완전 분류 [37개 레퍼런스 전수 분석]

> 스크립트 작성 시 같은 훅 유형을 반복하지 않도록 반드시 참고할 것.
> 제품별 3편 세트 내에서 G-1~G-7 중 서로 다른 유형 3개를 조합해야 한다.

### G-1. 훅 유형별 복사 가능 문장 템플릿

> `[대괄호]` 부분만 제품·증상에 맞게 교체. 나머지 문장은 그대로 사용.

---

#### 유형 1 — 질문형 (비율 16%)
구조: 증상/행동 + ? (짧고 직접적인 의문문)

**레퍼런스 원문:**
- "공부하다 말고 자리이탈?"
- "조금만 마음에 안 들어도 편식하는 아이?"
- "산만함의 원인이 스트레스라는 점 알고 계셨나요?"
- "아이 오메가3 좋다는 건 알겠는데 어떻게 골라야 하는지 모르겠는 분들?"

**문장 템플릿 (교체 후 그대로 사용):**
```
[아이 행동/증상]하는 아이?
[아이 행동] 하는 거 왜 그런지 알고 계셨나요?
[기존 시도]는 알겠는데, 어떻게 [해결]해야 하는지 모르겠는 분들?
```

---

#### 유형 2 — 충격 단언형 (비율 14%)
구조: 강한 단정문, 짧고 직접적, "~거임" 또는 단언 어투

**레퍼런스 원문:**
- "다크서클이 있는 아기들은 면역 박살난 거임"
- "밥만 먹으면 배가 아프다는 아이. 저희 아이가 몇 달 동안 그랬거든요."
- "유해균의 95%는 대장에 몰려 있어요."
- "아이 면역력의 70%는 장에서 시작해요."

**문장 템플릿:**
```
[증상]이 있는 아이들은 [상태] 박살난 거임.
[증상]하는 아이. 저희 아이가 [기간] 동안 그랬거든요.
[문제의 N%]는 [위치/원인]에 있어요.
```

---

#### 유형 3 — 금지·경고형 (비율 19%, 가장 높음)
구조: "~하면 안 된다" / "큰일 나요" / "먹지 마세요"

**레퍼런스 원문:**
- "이런 오메가3 먹으면 큰일 난다"
- "유산균 먹지 마세요!"
- "7년차 약사가 절대 안 먹이는 아이 오메가3"
- "이거 방치하면 절대 안 되는데요"
- "공유하시면 안돼요, 이거"

**문장 템플릿:**
```
이런 [제품명] 먹으면 큰일 난다.
[제품명] 먹지 마세요! (단, 이런 조건이라면)
[직함/경험] [화자]가 절대 안 먹이는 [제품명].
[증상] 방치하면 절대 안 되는데요.
```

---

#### 유형 4 — 실패고백형 (비율 19%, 공동 1위)
구조: "다 해봤는데" / "반신반의하면서" / "솔직히 몰랐거든요"

**레퍼런스 원문:**
- "유명한 거 다 사봤거든요. 퉤..."
- "솔직히 말하면 이게 무슨 예민함을 바꿔주나 싶었거든요"
- "사실 아이들 오메가3 비려서 잘 안 먹잖아요. 그래서 반신반의하면서 줬는데"
- "아무리 유명하고 좋은 [제품] 다 먹여봐도 아이가 거부하면 소용없겠죠?"

**문장 템플릿:**
```
[제품 카테고리] 유명한 거 다 사봤거든요. [실패 표현(퉤/별로/효과없어)]...
솔직히 이게 무슨 [기대 효과]를 바꿔주나 싶었거든요.
사실 아이들 [제품/성분] [단점]이라 잘 안 먹잖아요. 그래서 반신반의하면서 줬는데.
```

---

#### 유형 5 — 목록 나열형 (비율 14%)
구조: "1. ~ 2. ~" / "딱 N가지만"

**레퍼런스 원문:**
- "1. 아이 배가 빵빵하다 / 2. 설사나 변비가 반복된다 / 3. 아랫배에 가스가 찬다"
- "1. 다크서클이 있다 / 2. 맑은 콧물이 잦다"
- "딱 4가지만 체크하세요"
- "아랫배 가스, 변 닦을 때 묻어나옴 — 이런 사람은 유산균 먹지 마세요"

**문장 템플릿:**
```
1. [증상 A] / 2. [증상 B] / 3. [증상 C] — 해당하면 읽어보세요.
딱 [N]가지만 체크하세요.
[증상 A], [증상 B] — 이런 아이에게는 [제품/성분] 먹이면 안 돼요.
```

---

#### 유형 6 — 인사·자연 전환형 (비율 8%)
구조: 일상 대화나 아이 반응 먼저 → 자막으로 정보 전달 (나레이션 자체는 UGC)

**레퍼런스 원문 (나레이션/대화):**
- "이게 젤 맛있어!!" (아이 반응) → 자막: "우리 아이가 처음으로 먼저 찾은 영양제"
- "잘생겨요" / "쭝! 귀여워" → 자막: "성장기 두뇌발달에 가장 필수적인 DHA"
- "안녕하세요, 잠시만 저 똥 좀 싸고요" → 이후 고백 구조로 전환

**문장 템플릿:**
```
(아이 반응 자막) "[긍정 반응/먹는 장면]" → 자막1: "[성분/효과 정보]" → 자막2: "[제품명]"
(화자 일상 도입) "안녕[하세요], [의외의 상황 발언]" → 이후 고백 구조 전환
```

---

#### 유형 7 — Q&A 포맷형 (비율 5%)
구조: "Q. 증상 → A. 성분/제품"을 2~3회 반복

**레퍼런스 원문:**
- "Q. 아이가 감정 기복이 크고 예민해요 → 테아닌 드세요"
- "Q. 잠드는 데 오래 걸려요 → 트립토판 드세요"
- "Q. 긴장 많이 해요 → 비타민B 드세요"

**문장 템플릿:**
```
Q. [아이 증상 1] → [성분명] 드세요.
Q. [아이 증상 2] → [성분명] 드세요.
Q. [아이 증상 3] → [성분명] 드세요.
(이 모든 성분을 한 번에 챙기는 방법 = [제품명])
```

### G-3. 어미·말투 패턴 — 레퍼런스에서 실제 사용된 구체 형태

|---|---|---|
| **~거든요** | 이유 설명, 친근한 고백 | "유명한 거 다 사봤거든요" / "4달 됐거든요?" |
| **~더라고요/더라구요** | 발견·증언 | "집중력도 달라지더라구요" / "잘 먹더라고요" |
| **~잖아요** | 공통 전제·공감 유도 | "비려서 잘 안 먹잖아요" / "생선 챙기기 힘들잖아요" |
| **~거임/~임 (단언)** | 강한 경고, 충격 | "면역 박살난 거임" / "대장 다 망가진 거" |
| **~인데요** | 부드러운 제품 소개 | "뉴턴 젤리인데요" / "오메가3인데요" |
| **~았는데 (과거 반전)** | 오해→반전 | "예민한 줄 알았는데" / "꾀병인 줄 알았는데" |
| **? (짧은 질문)** | 즉각 공감 유도 | "자리이탈?" / "편식하는 아이?" |

> ⚠️ 한 세트(3편) 내에서 ~더라고요, ~거든요를 각각 훅 문장에 2회 이상 사용하지 않는다.

## H. 콘텐츠 유형별 나레이션 흐름 구조 — 문장 템플릿 [37개 전수 분석]

> 각 구조별로 레퍼런스 원문과 교체 가능한 템플릿을 함께 제공.
> `[대괄호]` 부분만 제품·증상에 맞게 교체. 나머지 문장 구조는 그대로 사용.
> 같은 제품 세트 내에서 동일 구조를 반복하지 않는다.

---

### H-1. 오가닉형 나레이션 흐름

오가닉은 **화자의 실제 경험**이 중심이다. 광고 티가 나면 안 된다.

---

#### 오가닉-A: 증상묘사 → 나의 오해 → 반전(진짜 원인) → 해결 → 변화
레퍼런스: 울보아들, 도윤맘, 편식

**레퍼런스 원문 흐름:**
> "사소한 일에도 우는 아이. 저도 처음엔 그냥 예민한 아이인 줄만 알았거든요. 근데 알고 보니 원인은 심리적 스트레스였대요. 스트레스 낮춰주는 루틴이 필요하다는 걸 그때 알았어요. [제품명] 챙겨주고 나서 많이 달라지더라고요."

**문장 템플릿:**
```
훅:   "[아이 증상/행동 묘사]하는 아이."
문장2: "처음엔 그냥 [흔한 오해]인 줄만 알았거든요."
문장3: "근데 알고 보니 원인은 [진짜 원인]이었대요."
문장4: "[해결 방법]이 필요하다는 걸 그때 알았어요."
결론:  "[제품명] 챙겨주고 나서 [변화 결과]더라고요."
```

**핵심 문장 패턴:** `~인 줄만 알았거든요 → 알고 보니 ~이었대요`

---

#### 오가닉-B: 실패경험 고백 → 이번엔 달라(반전) → 아이 반응 증언 → 제품 특징 → 결론
레퍼런스: 퉤, 찾아먹어, 잘시간

**레퍼런스 원문 흐름:**
> "유명한 거 다 사봤거든요. 퉤... 입에 넣자마자 뱉더라고요. 근데 이건 관심을 보이더니 스스로 꺼내 입에 넣고, 비려 안 비려? 안 비려. 젤리처럼 맛있다고 잘 먹더라고요. 비린 맛 없고 하루 1캡슐. 오메가3 전쟁 드디어 끝났어요."

**문장 템플릿:**
```
훅:   "[제품 카테고리] 유명한 거 다 사봤거든요. [의성어/실패 표현]..."
문장2: "[거부 반응 묘사]. 입에 넣자마자 뱉더라고요."
문장3: "근데 이건 [긍정 반응]을 보이더니 [변화 행동]."
문장4: "[제품 특징 1]에 [제품 특징 2]."
결론:  "[제품 카테고리] 전쟁 드디어 끝났어요."
```

> "입소문 나면서 하루 주문량 2천건 넘겼습니다. / 송가인님도 필수로 챙겨요. 장 문제의 90%는 대장에서, 유산균은 소장까지만이라 효과가 없어요. 이건 455억 마리 비피더스균만 넣었어요. 대량 생산이 어려워 조기 품절 예상입니다. 현재 할인가는 앞으로 만나보실 수 없습니다."

**문장 템플릿:**
```
훅:   "입소문 나면서 하루 주문량 [N]건 넘겼습니다." 또는 "[유명인]도 필수로 챙겨요."
문장2: "[문제의 N%]는 [위치]에서, [기존 방법]은 [한계 이유]라 효과가 없어요."
문장3: "이건 [핵심 성분 + 수치]만 넣었어요."
문장4: "[생산 제한 이유]라 조기 품절 예상입니다."
결론:  "현재 할인가는 앞으로 만나보실 수 없습니다."
```

**핵심 문장 패턴:** `숫자 소셜프루프 → 기존 방법 한계 → 제품 차별점 → 품절 긴박감 → 가격 CTA`

#### 커머셜-B: 대상 한정+효과 숫자 → 긴박감(방치하면) → 진짜 원인 → 제품 → 가격+CTA

**문장 템플릿:**
```
훅:   "[타겟 조건]이라면 한 가지를 꼭 기억하세요."
문장2: "단 [N일/기간]만 [섭취]해도 [효과 비유]처럼 [변화]."
문장3: "[골든타임/한계 나이/시기]를 넘어가면 [위험한 결과]가 [발생]합니다."
문장4: "[제품 독점 성분 or 최대 함량] [제품명]."

```

**핵심 문장 패턴:** `[타겟] 이라면 → N일만 먹어도 → [나이/시기] 넘으면 늦다 → [독점성분] → N% 더 싸다`

#### 커머셜-C: 감정적 반응(웃음/놀람) → 솔직한 고백(회의감) → 효과 묘사 → 가격 강조 → CTA

**문장 템플릿:**
```
훅:   "[함께 있는 사람]이랑 엄청 [감정 표현]했어요."
문장2: "솔직히 저도 [회의적 생각]이면 이렇게까지 안 샀죠."

문장4: "[할인 강도], 어떻게 이걸 참는데요."

```

**핵심 문장 패턴:** `감정 도입 → 솔직히 효과 없으면 안 샀죠 → 근데 느껴져서 → 할인 참을 수 없다 → 도전해보세요`

## I. 3편 세트 다양성 조합 강제 체크표

> 스크립터는 3편 작성 전 아래 표를 채워서 중복이 없는지 확인한다.
> 같은 칸에 동일 항목이 들어가면 반드시 변경한다.

### I-1. 세트 조합 확인표 (작성 전 필수)

| 편수 | 콘텐츠 유형 | 훅 유형 (G-1) | 나레이션 흐름 (H) | 화자 | 비주얼 시작 방식 |
|---|---|---|---|---|---|
| 1편 | | | | | |
| 2편 | | | | | |
| 3편 | | | | | |

**동일 금지 체크 항목:**
- 3편 내 같은 훅 유형 2회 이상 → 금지
- 3편 내 같은 나레이션 흐름 → 금지
- 3편 내 "아이 클로즈업 + 엄마 고백 2문장" 패턴 2회 이상 → 금지
- 3편 내 훅 문장에 ~더라고요 또는 ~거든요 2회 이상 → 금지

### I-2. 예시 조합표 — 뉴턴젤리 3편

| 편 | 유형 | 훅 | 흐름 | 화자 | 비주얼 |
|---|---|---|---|---|---|
| 1편 | 오가닉 | 인사·자연 전환형 | 오가닉-G (일상 UGC) | 아이+엄마 대화 | 아이 반응 먼저 |
| 2편 | 정보형 | Q&A 포맷형 | 정보형-F (Q&A) | 나레이터 | 화자 얼굴 직접 |
| 3편 | 커머셜 | 충격 단언형 | 커머셜-C (고백→효과→가격) | 엄마 1인칭 | 제품 언박싱 |
